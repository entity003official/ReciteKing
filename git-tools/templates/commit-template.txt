# --- 提交类型说明 ---
# feat:     新功能
# fix:      修复bug  
# docs:     文档更新
# style:    代码格式调整
# refactor: 代码重构
# test:     测试相关
# chore:    构建过程或辅助工具变动
#
# --- 常用范围 ---
# web:      网页相关
# mobile:   移动应用相关
# data:     数据库/数据文件相关
# ui:       用户界面相关
# docs:     文档相关
#
# --- 示例 ---
# feat(web): 添加课程选择界面
# fix(ui): 修复移动端导航栏显示问题
# docs: 更新v2.0使用说明
